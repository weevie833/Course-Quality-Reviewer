As explained in Student Notes: About Badges (Micro-Credentials) in the Course Overview module, when you successfully complete this course with a B- (80%) or better, you will receive a notification in your UNH email account approximately 11 days after the end of the course.

You can access your badge: 

  • via the "Go to the Badge" button in the email,
  • via the Credentials link in the Canvas course navigation of the course in which you earned the badge, or
  • externally via the Canvas Credentials external website .

See the UNH Student Information page for details for each option. Badges can be shared on social media platforms such as LinkedIn, on a website, or an email signature.  You can download and print the badge as well.

Important note if you are logging into Canvas Credentials for the first time:

UNH is using the Canvas Credentials badging platform starting Fall 2024.  UNH is no longer using Credly.  The Canvas Credentials website is: https://unh.badgr.com

  1. If you are logging into Canvas Credentials for the first time, use the 1st option shown below to create a new account. When you set up your account, it is important to use the email addressed used in Canvas which is: [username]@unh.edu, for example, if your username is abc123, then your email address would be: abc123@unh.edu.  You will need to create a password as well.

  2. After you have set up your account, you can log into Canvas Credentials by using single sign-on (SSO) (Option 2) or using your email and password (Option #3).

  3. We highly recommend that you update your account to add a personal email address. After you graduate from UNH, you will be able to use your personal email to log into Canvas Credentials.
