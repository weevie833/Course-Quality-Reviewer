Overview Chapter 3: Project Management

This chapter defines project management and outlines effective processes for accomplishing this critical activity. It also clarifies the importance of project management and outlines elements that must be properly managed to maximize chances for project success.

Overview Chapter 10: System Acquisition and Development

Leaders must remain open to using different methodologies, avoiding the temptation to declare that only one methodology will work best in all situations. Just as a mechanic or a plumber knows when to use which wrench or which screwdriver, the best practice is to understand multiple methodologies, the needs of the project, and the resources available to complete the project, and then select the approach that can best complete the project most effectively.
