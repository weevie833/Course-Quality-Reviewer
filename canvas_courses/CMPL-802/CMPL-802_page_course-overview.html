Welcome to CMPL 802 - Managing Virtualization and Cloud Systems

Through theoretical and practical exercises, students design and configure cloud and virtual environments essential to managing cloud technologies. Students engage directly with foundational cloud concepts, virtual hardware, migration strategies, and networking to develop cloud security, identity management, storage, and performance management implementations. Emphasis is on employing automated processes in maintaining and optimizing cloud resources, and using platforms like AWS, Microsoft Azure, and Google Cloud Platform.

Additional Learning Opportunity

Within the Cengage MindTap course, each module includes a Capstone Project. While not required this semester, you may complete it voluntarily. Doing so will involve signing up for a cloud platform, such as AWS, Azure, or Google Cloud, which may require a credit card and could incur charges once the free usage limits are exceeded. Because of this, the Capstone Project is optional.

Course Learning Outcomes:

  1. Design and execute effective migration strategies for moving applications and data to a cloud computing environment according to project specifications and available infrastructure.
  2. Construct and manage cloud-based networks using advanced IP addressing, subnetting, and virtual networking technologies across major cloud platforms according to project specifications, feasibility, and other constraints.
  3. Configure and manage virtual machines according to project specifications and long-term implementation needs.
  4. Employ open-source and proprietary automation tools to streamline cloud operations according to project specifications and security standards.
  5. Apply industry-compliant Cloud Security Best Practice strategies for protecting virtual networks and data against common threats.
  6. Use industry-standard diagnostic methods to troubleshoot cloud connectivity and network issues using command-line interface (CLI) tools.
