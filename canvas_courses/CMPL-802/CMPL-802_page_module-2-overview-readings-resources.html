Overview

This module introduces students to virtual hardware, focusing on compute virtualization technologies that power cloud infrastructure. Students will learn how to optimize virtual machine (VM) resources, deploy VMs in cloud environments, and explore emerging technologies such as serverless computing and containers.

Why it’s important:
Virtual hardware is the backbone of cloud computing. By understanding how virtual machines and container technologies work, students gain practical knowledge that is essential for deploying scalable, efficient, and cost-effective cloud solutions in today’s IT environments.

  • Section 2-1: Virtualization Technologies
  • Section 2-2: Virtualized Processing and Memory
  • Section 2-3: VMs in the Cloud
  • Section 2-4: VM Alternatives

Module 2 Objectives:

  1. Explain compute virtualization technologies.
  2. Optimize sizing of VM resources.
  3. Deploy VMs in the cloud.
  4. Describe serverless computing and container technologies.
