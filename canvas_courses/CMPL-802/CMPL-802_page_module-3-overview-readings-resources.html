Overview

This module focuses on the process of moving applications, data, and services to the cloud. Students will explore the phases and strategies of cloud migration, learn about different migration types and tools, compare deployment testing methods, and understand how cloud environments support agility and business continuity.

Why it’s important:
As more organizations shift to the cloud, professionals must understand how to plan and execute successful migrations. This module equips students with the knowledge to manage transitions effectively, minimize disruptions, and ensure systems remain agile and resilient in the cloud.

  • Section 3-1: Migration Planning
  • Section 3-2: Migration Execution
  • Section 3-3: Deployment Testing and Validation
  • Section 3-4: Cloud Agility
  • Section 3-5: Planning for Problems

Learning Objectives:

  1. Describe cloud migration phases and strategies.
  2. Explain cloud migration types and tools.
  3. Compare deployment testing types.
  4. Explain agility in the cloud.
  5. Apply business continuity planning.
