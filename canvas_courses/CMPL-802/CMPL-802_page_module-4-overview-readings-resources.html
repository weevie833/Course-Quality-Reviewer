Overview

This module introduces students to networking concepts within cloud environments. It covers how traditional networking principles apply to the cloud and provides hands-on experience managing IP address spaces and configuring basic networking resources in major cloud platforms, including AWS, Azure, and Google Cloud Platform (GCP).

Why it’s important:
Networking is a core component of cloud infrastructure. Understanding how to manage network configurations across different cloud providers enables students to build secure, scalable, and efficient cloud environments—an essential skill for any aspiring cloud or IT professional.

  • Section 4-1: Networking Concepts in the Cloud
  • Section 4-2: IP Address Spaces
  • Section 4-3: Networking in AWS
  • Section 4-4: Networking in Azure
  • Section 4-5: Networking in GCP

Learning Objectives:

  1. Explain how common networking concepts apply to cloud networking.
  2. Manage IP address spaces in the cloud.
  3. Configure basic AWS networking resources.
  4. Configure basic Azure networking resources.
  5. Configure basic GCP networking resources.
