Overview

This module focuses on protecting cloud environments by addressing common security threats and best practices for securing networks, computing resources, and cloud-hosted data. Students will also learn how to identify and troubleshoot cloud security issues to maintain a strong security posture.

Why it’s important:
Security is a top priority in cloud computing. As organizations increasingly rely on cloud services, professionals must be able to safeguard systems and data from evolving threats. This module provides essential knowledge and skills to help students secure cloud infrastructure and respond effectively to security challenges.

  • Section 6-1: Cloud Security Configurations
  • Section 6-2: Virtual Network Security 
  • Section 6-3: Compute Security
  • Section 6-4: Data Security
  • Section 6-5: Troubleshooting Cloud Security

Learning Objectives:

  1. Describe common threats to and configurations for cloud security.
  2. Secure cloud networks.
  3. Secure cloud compute resources.
  4. Secure cloud-hosted data.
  5. Troubleshoot cloud security issues.
