Overview

This module provides an in-depth look at cloud storage technologies and services. Students will compare different storage types, explore techniques for optimizing storage, and learn how to configure commonly used cloud storage solutions. The module also covers backup strategies and storage security best practices.

Why it’s important:
Effective cloud storage management is essential for performance, cost-efficiency, and data protection. This module equips students with the skills to choose the right storage solutions, secure data, and support business continuity through proper backup and storage practices.

  • Section 8-1: Storage Types
  • Section 8-2: Storage Optimization Techniques 
  • Section 8-3: Cloud Storage Services
  • Section 8-4: Creating and Storing Backups
  • Section 8-5: Storage Security

Learning Objectives:

  1. Compare storage technologies.
  2. Explain storage optimization techniques.
  3. Configure common cloud storage services.
  4. Describe options for creating and storing backups.
  5. Identify storage security techniques.
