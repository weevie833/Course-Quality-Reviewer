Overview

This week, the topics for discussion are Intrusion Detection and Prevention Systems (IDPS) and cryptographic policies that are implemented in an enhanced security paradigm. The emphasis is placed on the roles of these technologies in safeguarding organizational resources against security risks, identifying intrusions, and maintaining data confidentiality in their movement from one point to the other. 

Readings

  • Module 9 - Security Technology: Intrusion Detection and Prevention Systems and Other Security Tools (47 pages)
  • Module 10 - Cryptography  (34 pages)
  • Evaluating modern intrusion detection methods in the face of Gen V multi-vector attacks with fuzzy AHP-TOPSIS (12 pgs)
  • A critical review of intrusion detection systems in the internet of things: techniques, deployment strategy, validation strategy, attacks, public datasets and challenges (27 pgs)
