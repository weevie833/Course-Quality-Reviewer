Each week, you will devote time to developing artifacts related to your proposed project that will be accumulated into a singular portfolio which you will submit in Module 15. The portfolio will include all of the documents that you will develop along the way:

  1. Phase 1: Project Proposal (Modules 1–2)

  4. Phase 4: Project Execution (Modules 7–8)

  6. Phase 6: Risk and Adaptability (Module 11)

Each Phase must include:

  • A detailed description of the tasks involved in that phase.
  • All associated files and documentation relevant to that phase.
  • Any additional requirements specified by the course instructor.
