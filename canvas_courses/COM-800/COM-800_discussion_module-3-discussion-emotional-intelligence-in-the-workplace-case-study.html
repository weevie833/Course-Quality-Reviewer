For this forum, you will be asked to create a case study of an organization that has a need to enhance its communications related to emotional intelligence. Be sure to read our resources this week before beginning so that you are using common language. Case studies are typically only a few paragraphs in length and include:

  • A description of the organization and its mission, vision, and values (ideally, you would model this on a real organization you have an interest in – whether it’s one you work for or want to work for; you may de-identify the name if you want to use your own workplace)
  • A description of the problem and consequences

 

Our responses will be located in the Module 3 Discussion: Case Study Recommendations.  Initial post is due by Thursday midnight and replies to others by Sunday midnight. When writing replies to others, please add a question to encourage a more interactive experience in the discussions. Make sure to review the Discussion Rubric (located in Course Resources).
