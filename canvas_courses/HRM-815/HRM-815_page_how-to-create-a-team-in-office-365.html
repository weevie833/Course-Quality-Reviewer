Canvas includes an integrated method for creating a Team in UNH's Office 365 system by using the Microsoft Education feature in the Canvas course navigation area. Follow these steps:

  1. Select Microsoft Education from the Course Navigation menu. 
  2. Select the "Teams - View and create teams" option



  3. Once the new Team is created, you will notice that the Team is not activated by default. Click the Open button to review the Team in UNH Office 365. The Team will named the same as the course.



  4. In UNH Teams, activate the class team when you are ready for students to access it.

If your class has more than one group, create individual channels for each group so that each group is private.

  1. Go to the designated class Teams area.
  2. Click the three-dot menu on the lower right corner of the Team icon and select Add Channel.
  3. Name the channel and choose Private in the "Choose a channel type" menu.
  4. Add students to the channel by typing in a name until it auto-discovers the person you want to add. Select the person, then repeat this process for all other group members.
  5. Repeat this process for creating additional channels for each group on the course.

Direct students to access their Teams environment by navigating from their UNH Outlook account:

  1. Log in to UNH Outlook email.
  2. Click the waffle menu in the header on the upper left.
  3. Select Teams from the app choices.
  4. By default, all Teams will display where the student is a member.
