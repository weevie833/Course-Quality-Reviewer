Overview

At the end of this module you should be able to:

  1. Explain why project management is crucial in today’s world.
  2. Define a project and differentiate projects from routine operations.
  3. Explain how projects are integral to actualizing organizational strategies.
  4. Articulate how different types of project management structures are more or less appropriate to use under different conditions.
  5. Describe how a project priority system optimizes available time and resources.
  6. Use financial and non-financial criteria to select projects for execution.
  7. Describe how a project provides value in the context of meeting organizational objectives.
