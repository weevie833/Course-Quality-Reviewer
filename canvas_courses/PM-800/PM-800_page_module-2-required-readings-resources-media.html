Overview

Last week you learned about the practices within project management. Now you will now learn how to define the project through the creation of the charter and scope documents and developing SMART goals. By creating a project charter, the project sponsor authorizes the identified project manager to apply organizational resources to begin planning the project. The charter provides the project manager with an explanation of the project’s scope, its expected outcomes and deliverables, time frames, and budgetary guidance.

At the end of this module you should be able to:

  1. Develop goals according to the SMART Goal model.
  2. Determine project requirements and prioritize them according to project expectations.
  3. Create a project charter and project scope document under the guidance of the project sponsor.
