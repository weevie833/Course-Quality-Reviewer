Overview

One of the challenges project managers face is to complete a project within an approved budget.  After estimating all project costs and determining the potential impact of risks, the project manager can now develop the budget and obtain approval for it.  The project manager develops a cost management plan that defines how to estimate costs, determine the budget, and define the procedures to monitor and control the costs over the life of the project.  A commonly used method to monitor costs relative to completed work is through the application of the Earned Value Method (EVM).

At the end of this module, you should be able to:

  1. Be able to calculate and interpret different project performance indices
  2. Be able to develop a project reporting system that will ensure project control and progress
  3. Develop a detailed cost estimate and baseline for your project.
  4. Employ the cumulative cost curve as a planning tool for your project.
  5. Determine the project budget by adding the management reserve to the cost baseline.
  6. Develop the Cost Management Plan for your project.
