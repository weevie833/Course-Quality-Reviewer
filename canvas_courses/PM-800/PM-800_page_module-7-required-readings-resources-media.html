Overview

At the end of this module you should be able to:

  1. Create an awareness of the network of relationships that need to be managed to be a successful project manager
  2. Understand and discuss the importance of building trust for project success
  3. Identify some of the qualities of an effective project manager
  4. Understand the importance of the project manager's behavior in shaping how a team performs.
