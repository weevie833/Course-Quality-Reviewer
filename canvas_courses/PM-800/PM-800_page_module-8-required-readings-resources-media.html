Overview

At the end of this module course you should be able to:

  1. Articulate the value of conducting a project review as part of the project closure process.
  2. Describe how each major element of a project review provides value for continuous improvement.
  3. Explain the key issues surrounding project closure that impact both product and process for this project and future projects.
  4. Evaluate how lessons learned in a project inform planning for future projects.
