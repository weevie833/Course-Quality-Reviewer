Welcome to PM811: Project Chartering and Planning!

In this course, you will start by learning how a project charter refines a project idea in a more concrete project narrative. You will develop a project’s work breakdown structure and establish a realistic project schedule and budget.  You will conduct project risk identification and assessment sessions, to evaluate the overall risk posture of the project. You will also define the communications and change control plan. Within this course, you will have the opportunity to practice the various methods and tools used in a successful project, in preparation for doing the same within your capstone projects.

Please note that Project Quality Management, Project Procurement Management and Human Resource Planning will be covered in greater detail in future courses.
