Overview

Projects are undertaken to achieve defined goals by producing deliverables. Portfolio managers select, prioritize, and allocate resources to projects.

  1. At the end of this module you should be able to:
  2. Understand the need for a project priority system
  3. Use financial and non-financial criterion to select projects for execution
  4. Understand the importance of being aligned to the organization's objectives and delivering value
  5. Understand the culture and organizational structure
