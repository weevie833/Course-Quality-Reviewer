Overview

Last week you learned about project selection techniques. Now you will now learn how to define the project through the creation of the charter and scope documents and developing SMART goals. You will also work on developing the Project Change Management Plan (not to be confused with organizational change management - OCM). You will use the planning and execution of a party as the basis to develop project artifacts through this course and continue all the way through the end of the project in the next course PM813.

At the end of this module you should be able to:

  1. Understand and develop SMART goals
  2. Determine and prioritize requirements
  3. Create a project charter & project scope document
  4. Understand the various sources of change requests such as defect repair, corrective action etc
  5. Determine the change control plan for your project
