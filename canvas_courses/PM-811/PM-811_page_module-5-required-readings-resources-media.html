Overview

One of the challenges that project managers face is to complete a project within the approved budget. To achieve this, they must develop a cost management plan that defines how to estimate costs, determine the budget and define the procedures to monitor and control the costs.

At the end of this module you should be able to:

  1. Determine the estimating technique to be used for the project
  2. Develop the detailed cost estimate for your project
  3. Develop the Cost Management Plan for your project (draft)
