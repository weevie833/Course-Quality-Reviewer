Overview

After the successful completion of project planning activities, the project manager prepares to lead the team through the execution of the project. This implies integration, which includes but is not limited to allocating resources, evaluating and resolving competing demands, customizing approaches to meet project goals and delivering business value etc. In addition, companies are constantly balancing scarce resources, competition changing scenarios etc. In this module you will prepare the project management plan that defines how the project is executed, monitored, controlled, and closed.

At the end of this module you should be able to:

  1. Describe the contribution Project integration Management provides business value and achievement of project objectives.
  2. Review planning artifacts created in PM811 and develop the project management plan
  3. Describe the functional relationships between projects, project management and achieving organizational objectives.
  4. Explain how the role of the project manager relates to delivering business value, both directly and indirectly.
