Overview

Once a project plan is approved and the project execution begins, the project manager needs to focus on controlling the project and reporting status to senior managers who perform governance.  The project manager needs to control all aspects of the project: scope, cost, schedule, quality, resources, risk, etc. and forecast expenditures.

At the end of this module you should be able to:

  1. Associate the basic components of Earned Value Management (EVM) to the business value of project management.
    • Earned Value (EV)
    • Planned Value (PV)
    • Actual Cost (AC)
    • Budget at Completion (BAC)
  2. Associate the status components of EVM to the business value of project management.
    • Cost Variance (CV)
    • Schedule Variance (SV)
    • Cost Performance Index (CPI)
    • Schedule Performance Index (SPI)
  3. Associate the forecasting elements of EVM to the business value of project management.
    • Estimate at Completion
    • Variance at Completion
    • To-complete Performance Index
  4. Analyze the calculations and assess the project
  5. Be able to recommend corrective action where necessary
