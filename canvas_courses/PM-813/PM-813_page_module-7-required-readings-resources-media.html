Overview

There is a lot of variation in project work from the easily definable to the ambiguous exploratory types. This variety in projects led to the development of alternate project management methods such as Agile.

At the end of this module you should be able to:

  1. Assess which projects are better suited to Agile vs traditional project management methods.
  2. Describe how core Agile principles provide operational efficiencies and business value to a project.
