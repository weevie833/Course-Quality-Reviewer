Overview

The previous module introduced you to the iterative development process in Agile that requires active customer participation and an agile mindset. In this module, we will delve deeper into implementing Agile.

At the end of this module you should be able to:

  1. Evaluate and select an Agile environment option for a project.
  2. Propose how to deliver in an Agile environment.
  3. Propose best ways to integrate Agile methods in your projects
