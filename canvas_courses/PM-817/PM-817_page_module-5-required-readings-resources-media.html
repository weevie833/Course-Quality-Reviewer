Overview

A key process within PPM is the prioritization of project requests, the balancing of requests for new projects against projects already underway. We will look at various decision criteria, including resource allocations, that influence those prioritization decisions.  

Learning Objectives:

  1. Define impartial prioritization criteria
  2. Balance resource allocation across the project portfolio
  3. Select and prioritize projects in the portfolio
