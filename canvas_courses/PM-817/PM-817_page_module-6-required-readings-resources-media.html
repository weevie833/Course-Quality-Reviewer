Overview

In this module, we further expand the discussion of maintaining a portfolio that is supportive of organizational objectives. We will address the topic of key performance indicators (KPIs) for the portfolio and how to communicate portfolio performance to executives.    

Learning Objectives:

  1. Balance resource allocation across the project portfolio
  2. Select and prioritize projects in the portfolio based on resource availability
  3. Conduct a project portfolio review with executives
