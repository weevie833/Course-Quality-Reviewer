Overview

The role of a project manager in information technology (IT) projects can be very complex, reflecting the complexity of the projects being undertaken. The successful IT Project Manager is expected to have not only a demonstrated capability of managing projects and the business acumen to understand the relationship of the project to the overall organizational objectives. They should also have some working knowledge of the different disciplines within an IT project.  That knowledge should encompass abilities such as the ability to determine the appropriate software development life cycle for the project, validate a network architecture design, and integrate business analysts, developers, quality control specialists, and operations staff into a fully integrated project team. They should have a working familiarity with the various principles and concepts associated with software engineering.

Upon completion of this module, students will be able to identify those elements of managing an IT project that are unique to these projects. They will be able to identify the various organizations that need to be engaged, and why, with the project to ensure project success.
