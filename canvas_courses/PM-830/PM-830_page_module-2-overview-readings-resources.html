Overview

One of the more useful tools an IT PM uses when planning an IT project is a life cycle model. This week students will learn about the multitude of life cycle models that are applied within business. These include various forms of project management life cycle models, software development life cycle models, system engineering life cycle models, and product management life cycle models.  A representation of each will be analyzed along with the scenarios within which these models could be applied, and how they are applied.

Students will determine how these models can be integrated to complement each other, resulting in a custom life cycle model for the project at hand.
